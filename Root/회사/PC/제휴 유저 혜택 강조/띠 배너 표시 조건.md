1. 제휴유저 아닐때 띠배너 없어야함
2. 전문가 찾기
	1. 배열에 일치안하면 상품없다는 배너
	2. 배열에 일치하고 
		1. 회기수 없으면 회기수 없다는 배너
		2. 회기수 있으면  상품 있다는 배너
3. 전문가 상세
	1. 빈배열이면 상품 없다는 배너
	2. 배열 있고
		1. 회기수 없으면 회기수 없다는 배너
		2. 회기수 있으면  상품 있다는 배너
4. 바우처
	1. /sale-template-sellers?seller_id api 쳐서 is_available_use_eap_coupon가 1인 상품이 존재하는 상품목록이 하나도 없는 경우 상품 없다는 배너
	2. 회기수 없으면 회기수 없다는 배너
	3. 회기수 있으면 상품 있다는 배너
